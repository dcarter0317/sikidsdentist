# SI Kids Dentist

A modern web project for SI Kids Dentist.

## Project Structure

- `index.html` — Main HTML file
- `script.js` — JavaScript logic
- `style.css` — Main stylesheet
- `assets/` — Images, fonts, and other static assets
    - `michelle_flanigan.css` — Additional CSS
    - `fonts/` — Font files
    - `stitch_dark_svg_converter/` — SVG assets

## Getting Started

1. Clone the repository:
   ```sh
   git clone <repo-url>
   ```
2. Open the project folder in your editor.
3. Open `index.html` in your browser to view the site.

## Development

- Edit `script.js` for JavaScript changes.
- Edit `style.css` or `assets/michelle_flanigan.css` for CSS changes.
- Add images, fonts, or SVGs to the `assets/` directory as needed.

## Best Practices

- Keep `.DS_Store` and other OS/editor files out of version control (see `.gitignore`).
- Use clear commit messages.
- Organize assets logically in the `assets/` folder.
